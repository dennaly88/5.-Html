Estructura General de HTML 5


<!DOCTYPE> Esquema General de HTML 5

<html > Atributo lang indica el idioma del contenido de la pagina

<head > Cabecera del documento HTML

<meta > Tipo de contenido y codificacion de caracteres

<title > Titulo de la pagina

<link > Enlace a la hoja de estilos CSS

<script > Enlace a un archivo JavaScript externo

<style > Estilos CSS internos

<base >URL base para los enlaces relativos

<body >Cuerpo del documento HTML

<header> Encabezado del documento HTML

<main> Contenido principal del documento HTML

<nav> Navegacion del documento HTML

<section> Seccion del documento HTML

<article> Articulo del documento HTML

<aside> Contenido adicional del documento HTML

< h1 h6> Encabezado de nivel 1 al 6

<footer> Pie de pagina del documento HTML

< p> Parrafo del documento HTML

< ul> Lista desordenada del documento HTML

< ol> Lista ordenada del documento HTML

< li> Elemento de lista del documento HTML

< a> Enlace del documento HTML

< img> Imagen del documento HTML

< table> Tabla del documento HTML

< tr> Fila de la tabla del documento HTML

< td >Celda de la tabla del documento HTML

< th> Encabezado de la tabla del documento HTML

< form >Formulario del documento HTML
