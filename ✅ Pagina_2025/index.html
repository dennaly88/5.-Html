HTML 5

HyperText Markup Language

HTML son las siglas de HyperText Markup Language, que se traduce al español como Lenguaje de Marcado de Hipertexto. Es el código que se utiliza para estructurar y mostrar una página web y su contenido. Por ejemplo, el contenido podría ser párrafos, una lista con viñetas o imágenes.Piensa en HTML como los "planos" o la "estructura" de una página web. No es un lenguaje de programación (no puedes hacer que las cosas "hagan" algo con HTML), sino un lenguaje de marcado que utiliza una serie de elementos para definir la forma en que se presenta el contenido en un navegador web.
